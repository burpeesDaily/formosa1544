An Python Coding Sample for Shun’s Vineyard¶

The Binary Trees library is an example to demonstrate how to use Sphinx to generate documents. Although this is just an example code, the sample code follows numpydoc docstring guide and My Python Coding Style and Principles.

Requirements¶

The Binary Trees library has the following dependencies:

  • Python 3.7+

  • Sphinx 2.2+

Installation¶

To download the Binary Trees library from Github, do:

$ git clone https://github.com/shunsvineyard/python-sample-code.git
